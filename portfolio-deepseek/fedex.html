Project
Project 2 : FedEx

Academic Project Disclosure

This design project was completed as part of my academic learning journey at IIT Roorkee, under the guidance and mentorship of Mr. Nishant Kumar, Territory Manager at FedEx Vadodara. This is an independent academic exercise and not an official FedEx project or endorsed by FedEx Corporation. No proprietary or confidential business data was used.

Overview

In the contemporary logistics ecosystem, territory managers operate within an increasingly fragmented information architecture. The challenge lies in synthesizing operational data into a single actionable surface.

Problem Statement

How might we design a unified dashboard interface that consolidates fragmented data streams into a cohesive analytical environment?

Final Design

The unified dashboard interface consolidates all critical territory management functions into a single workspace.

Conclusion

This project demonstrates how logistics data can be unified into a single analytical surface, enabling faster and more confident decision-making.
