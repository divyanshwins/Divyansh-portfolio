Arth : an E-commerce analytics dashboard

Target Users
Ecommerce executives, owners, analysts
Context
I was an analyst & consultant before
Year
2025
Project
Project 1 : ARTH

Overview

During my experience as a Growth consultant, I figured out the digital ecommerce space for independent ecommerce owners has a big gap of data unification. Multiple dashboards have multiple data centres which create different reports for profitability, as multiple data stakeholders exist in the market for marketing, logistics, and product.

Problem Statement

How might we create a dashboard which unifies the different avenues of data like logistics, marketing, and product SKU data at a single place, so that our customer gets holistic overview and reporting?

Stakeholder Interviews

I interviewed many stakeholders from my prior job and friends in this project, but I'm not mentioning their names due to confidentiality.

Key Insights

  • Types of report they need
  • Time duration and distribution they look for
  • Important analytical metrics
  • Critical analytical metrics
  • Troubleshooting flow

Design System

For this project, I used the Carbon Design System by IBM, a comprehensive design system specifically built for enterprise-grade applications.

Why Carbon for Analytics?

  • Built for Data-Heavy Applications
  • Robust Data Visualization Components
  • Clear Information Hierarchy
  • Enterprise-Grade Accessibility
  • Professional & Clean Aesthetic

Final Designs

Comprehensive dashboard views covering orders, profitability, and shipping analytics in one unified interface.

Conclusion

The Arth e-commerce analytics dashboard successfully unifies disparate data sources into a cohesive interface, providing business owners with the holistic view they need to make informed decisions.
